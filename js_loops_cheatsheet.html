Loops

Five ways to repeat code in JavaScript — pick based on what you're iterating

for
Known repetitions
Best when you know exactly how many times to loop. Three parts: init → condition → update.
for (let i = 0; i < 5; i++) { console.log(i); }
→ 0 1 2 3 4
while
Condition first
Checks condition before each iteration — may not run at all if condition starts false.
let i = 0; while (i < 3) { console.log(i); i++; }
→ 0 1 2
do...while
Runs at least once
Checks condition after running — guarantees the block executes at least one time.
let x = 10; do { console.log(x); } while (x < 5);
→ 10 (runs once, then stops)
for...of
Loop through values
Iterates over values — works on arrays, strings, Maps, Sets.
let nums = [10, 20, 30]; for (let num of nums) { console.log(num); } // works on strings too for (let ch of "hi") { console.log(ch); }
→ 10 20 30 / h i
for...in
Loop through keys
Iterates over keys of an object. Use bracket notation to get values.
let person = { name: "Sajin", age: 20 }; for (let key in person) { console.log(key, person[key]); }
→ name Sajin / age 20

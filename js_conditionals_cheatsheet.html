Conditionals — if / else / else if

Control which block of code runs based on a condition

if
Run if true
Executes the block only when the condition is truthy.
let age = 18; if (age >= 18) { console.log("Adult"); }
→ Adult
else
Fallback when false
Runs when the if condition is falsy.
let age = 15; if (age >= 18) { console.log("Adult"); } else { console.log("Minor"); }
→ Minor
else if
Multiple conditions
Chain additional checks — first truthy block wins.
let marks = 80; if (marks >= 90) { console.log("A"); } else if (marks >= 70) { console.log("B"); } else { console.log("C"); }
→ B
Truthy & Falsy
What counts as false?
These 7 values are falsy — everything else is truthy.
false0-0""nullundefinedNaN
"hello"1[]{}
if ("hello") { console.log("Runs"); }
→ Runs
Nested if
if inside an if
Nest an if block inside another to check multiple conditions step by step.
let age = 20; let hasID = true; if (age >= 18) { if (hasID) { console.log("Allowed"); } }
→ Allowed
Logical operators
Combine conditions
Use inside if to combine multiple checks without nesting.
&& both true || either true ! not
if (age >= 18 && hasID) { console.log("Allowed"); }

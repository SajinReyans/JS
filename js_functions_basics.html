Function declaration, parameters & return

How to define functions, pass data in, and send data back

Declaration
Define with function keyword
Use the function keyword, give it a name, then call it by that name.
function greet() { console.log("Hello"); } greet();
→ Hello
Hoisting
Call before defining
Function declarations are fully hoisted — you can call them above where they appear in the file.
sayHi(); // works! function sayHi() { console.log("Hi"); }
→ Hi
Parameters & Arguments
Pass data into a function
name is the parameter (placeholder). "Sajin" is the argument (real value).
function add(a, b) { console.log(a + b); } add(2, 3);
→ 5
Default Parameters
Fallback when missing
If an argument is not passed, the default value is used instead of undefined.
function greet(name = "Guest") { console.log(name); } greet(); greet("Sajin");
→ Guest / Sajin
Rest Parameters
Collect many args into array
Use ...name as the last parameter to collect any number of arguments into an array.
function total(...nums) { console.log(nums); } total(1, 2, 3);
→ [1, 2, 3]
Return
Send a value back
return sends data out of the function and also stops execution — nothing after it runs.
function add(a, b) { return a + b; console.log("never"); // skipped } let result = add(2, 3); console.log(result);
→ 5

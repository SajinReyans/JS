Ternary operator & loop control

Shorthand conditionals and ways to exit or skip inside loops

Ternary
One-line if/else
Returns one of two values based on a condition. Shorter than writing a full if/else.
// condition ? valueIfTrue : valueIfFalse let age = 20; let result = age >= 18 ? "Adult" : "Minor"; console.log(result);
→ Adult
Nested ternary
Chain conditions
A ternary can be the false-branch of another. Use sparingly — hard to read with more than 2 levels.
let score = 85; let grade = score > 90 ? "A" : score > 70 ? "B" : "C"; console.log(grade);
→ B
break
Exit the loop
Immediately stops the loop and jumps to the code after it.
for (let i = 0; i < 10; i++) { if (i === 3) break; console.log(i); }
→ 0 1 2 (stops at 3)
continue
Skip this iteration
Skips the rest of the current iteration and jumps to the next one.
for (let i = 0; i < 5; i++) { if (i === 2) continue; console.log(i); }
→ 0 1 3 4 (skips 2)

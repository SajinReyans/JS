Switch statement

Cleaner alternative to long if / else if chains — matches a value to a case

switch
Basic usage
Evaluates a value and jumps to the matching case. default runs if nothing matches.
let day = 2; switch (day) { case 1: console.log("Mon"); break; case 2: console.log("Tue"); break; default: console.log("Invalid"); }
→ Tue
break
Why it matters
Without break, execution falls through into the next case automatically.
let x = 1; switch (x) { case 1: console.log("A"); // no break! case 2: console.log("B"); }
→ A B (both run!)
Fall-through
Intentional grouping
Stack multiple cases with no code between them to share one action — intentional fall-through.
switch (day) { case 1: case 2: case 3: console.log("Weekday"); break; case 6: case 7: console.log("Weekend"); break; }
default
Catch-all case
default is like else — runs when no case matches. No break needed if it's last.
switch (color) { case "red": console.log("Stop"); break; case "green": console.log("Go"); break; default: console.log("Unknown"); }

Comments

Three types of comments in JavaScript — none are executed by the engine

Single line
Quick inline notes
Start with // — everything after it on that line is ignored.
// calculate total price let total = price * qty;
Multi line
Block comments
Open with /* and close with */ — spans as many lines as needed.
/* This section handles user authentication */ function login() {}
JSDoc
Documentation comments
Starts with /** — used by editors to show tooltips and auto-generate API docs.
/** * Adds two numbers * @param {number} a * @param {number} b * @returns {number} */ function add(a, b) {}

Arrays — create, access & mutate

How to create arrays, read and update elements, and add or remove items

Create
Three ways to make an array
// literal — most common let arr = [1, 2, 3]; // empty let arr = []; // constructor — avoid let arr = new Array(1, 2, 3);
Access & modify
Read and change by index
Arrays are zero-indexed. Use bracket notation to read or overwrite any position.
let nums = [10, 20, 30]; nums[0]; // 10 nums[1] = 99; // modify nums.length; // 3 nums[nums.length-1];// last
Add elements
push & unshift
push()add to end
unshift()add to beginning
let arr = [1, 2, 3]; arr.push(4); // [1, 2, 3, 4] arr.unshift(0); // [0, 1, 2, 3, 4]
Remove elements
pop & shift
pop()remove from end
shift()remove from beginning
let arr = [1, 2, 3]; arr.pop(); // removes 3 arr.shift(); // removes 1 // [2]

Array search & test methods

Find elements, check positions, and test whether conditions pass

find()
First matching value
returns value
Returns the first element where the callback is true. Returns undefined if nothing matches.
[1, 4, 7] .find(n => n > 3);
→ 4 (first match)
findIndex()
First matching index
returns index
Same as find() but returns the position instead of the value. Returns -1 if not found.
[1, 4, 7] .findIndex(n => n > 3);
→ 1 (position of 4)
some()
At least one passes?
returns boolean
Returns true if at least one element satisfies the condition. Short-circuits on first match.
[1, 2, 3] .some(n => n > 2);
→ true
every()
All elements pass?
returns boolean
Returns true only if every element satisfies the condition. Short-circuits on first failure.
[2, 4, 6] .every(n => n % 2 === 0);
→ true
includes()
Does value exist?
returns boolean
Simple existence check — no callback needed. Uses strict equality.
[1, 2, 5].includes(5); [1, 2, 5].includes(9);
→ true / false
indexOf()
Position of a value
returns index
Returns the first index of the value, or -1 if not found.
[10, 20, 30].indexOf(20); [10, 20, 30].indexOf(99);
→ 1 / -1

Array transform & utility methods

Sort, slice, splice, destructuring, spread, and other handy methods

sort()
Always pass a comparator
mutates original
Default sort is alphabetical (string-based) — always pass (a,b)=>a-b for numbers.
// wrong — string sort [1,10,2].sort();
→ [1, 10, 2] ✗
// ascending [1,10,2].sort((a,b) => a-b); // descending [1,10,2].sort((a,b) => b-a); // by object property users.sort((a,b) => a.age-b.age);
→ [1, 2, 10]
slice(start, end)
Extract without modifying
original unchanged
Returns a portion from start up to but not including end. Negative index counts from end.
[1,2,3,4].slice(1, 3); // start 1, up to (not inc) 3 [1,2,3,4].slice(-2); // last 2 elements
→ [2, 3] / [3, 4]
splice(start, del, ...items)
Remove, insert, replace
mutates original
// remove 2 from index 1 arr.splice(1, 2); // insert 99 at index 1 arr.splice(1, 0, 99); // replace 1 item at index 1 arr.splice(1, 1, 99);
Destructuring
Unpack into variables
Pull array values directly into named variables by position.
let [a, b] = [10, 20]; // a=10, b=20 // skip index 1 let [x, , z] = [1, 2, 3]; // x=1, z=3 // rest of array let [first, ...rest] = [1,2,3]; // first=1, rest=[2,3]
Spread operator
Copy and merge arrays
Use ... to expand an array into individual elements — copy, merge, or pass as args.
// copy let copy = [...arr]; // merge let merged = [...a, ...b]; // pass as args Math.max(...nums);
Utility methods
concat, reverse, join, flat, from
concat(b)merge a + b → new array
reverse()flip order (mutates)
join("-")array → string with separator
flat(depth)flatten nested arrays
Array.from()convert iterable to array
isArray()check if value is array
[1,2,3].join("-"); // "1-2-3" [1,[2,[3]]].flat(2); // [1,2,3] Array.from("hi"); // ["h","i"]
